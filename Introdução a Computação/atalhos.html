Atalhos no vscode

Para comentar em qualquer linguagem: Shift + Alt + A

Para adicionar uma quantidade especificas de li: li*numerodeitens | ou digitar e quando for evelopar digita li* e dar enter

Para selecionar várias linhas/tags iguais: ctrl + D

Para construir varias Tags de uma vez só, um dentro da outra: usar o '>' , ex: header>nav>ul>(li>a)*4

Para construir varias Tags de uma vez só, uma seguida da outra: usar o '+' , ex: h1+h2+p

Também consigo misturar, exemplo: (header>nav>ul>(li>a)*4)+(main>(h1+p))+(footer.meufooter>p#meu-paragrafo>p)
